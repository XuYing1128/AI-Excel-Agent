# 表格类型 Playbook

## personal_budget

输入区：日期、分类、项目、预算、实际。计算区：差异、完成率。汇总：预算合计、实际合计、结余。

## quotation

输入区：产品、规格、数量、单价、折扣、税率。计算区：未税金额、税额、含税总价。汇总：小计、税额、总价。

## inventory

输入区：SKU、品名、期初、入库、出库、安全库存。计算区：期末库存、补货状态、周转率。

## sales_report

输入区：日期、区域、产品、品类、数量、单价、成本。计算区：销售额、毛利、毛利率。汇总：按品类/区域/月度趋势。

## ecommerce_analysis

输入区：订单、SKU、渠道、数量、单价、退款、成本。计算区：GMV、净销售、毛利、退款率、ROI。

## project_plan

输入区：任务、负责人、开始、结束、完成率。计算区：工期、状态、是否延期。

## schedule

输入区：日期/星期、时段、课程或班次、负责人、地点。计算区：时长、排班次数。

## attendance

输入区：员工、日期、应出勤、实出勤、迟到分钟。计算区：出勤率、状态。

## finance_model

输入区：月份、收入、成本、费用。计算区：毛利、净利、毛利率、净利率、ROI。

## dashboard

源数据 sheet 保存明细，Dashboard sheet 放 KPI、趋势、Top N 和图表。
